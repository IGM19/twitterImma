# [Immagine] 🚀

Una red social minimalista construida con Vanilla HTML, CSS y JavaScript, con el poder de Supabase.

## 🛠️ Configuración Local

1. Clona este repositorio.
2. Abre `index.html` en tu navegador.
3. Asegúrate de que `config.js` tenga tus credenciales de Supabase.

## ☁️ Despliegue en Cloudflare Pages

1. **Sube tu código a GitHub**:
   ```bash
   git add .
   git commit -m "Despegue inicial"
   git push origin main
   ```
2. **Conecta con Cloudflare**:
   - Ve al panel de [Cloudflare Pages](https://dash.cloudflare.com/).
   - Selecciona **"Connect to Git"**.
   - Elige este repositorio.
   - En **Build settings**, deja todo por defecto (Build command: vacío, Build output directory: raíz `/`).
3. **¡Listo!** Tu app estará live en segundos.

## 📝 Requisitos de Base de Datos (Supabase)

Ejecuta este SQL en tu panel de Supabase para que todo funcione:

```sql
create table posts (
  id uuid default gen_random_uuid() primary key,
  content text not null,
  author_id text not null,
  created_at timestamptz default now()
);

create table likes (
  id bigint generated by default as identity primary key,
  post_id uuid references posts(id) on delete cascade not null,
  device_id text not null,
  unique(post_id, device_id)
);

alter table posts enable row level security;
alter table likes enable row level security;

create policy "Ver posts" on posts for select using (true);
create policy "Crear posts" on posts for insert with check (true);
create policy "Borrar mis posts" on posts for delete using (true);
create policy "Ver likes" on likes for select using (true);
create policy "Dar like" on likes for insert with check (true);
```
